Features

Here are all the things I used to build the demo (frontend and backend included).

    Backend
  • Python - my favorite language to code in, used to develop backend.
  • Flask - super fast development and deployment backend framework
  • TinyDB - a Mongo-like document-based internal Python database that uses JSON and query functions
  • Email Validator python module used for validating email addresses and manually for the rest
  • Flask flash messages used to send server status messages after certain interactions (and errors)
  • Using Dad-Jokes API from ICanHazDadJoke for humorous jokes

    • Frontend
  • Meticulously designed with Brand and Color identity in mind
  • Desktop and Mobile ready Responsive design
  • Bootstrap 4.4 framework used for fast polished intuitive CSS classes
  • Font Awesome icon framework used for extra UI flair
  • Illustrations used are from the open source UnDraw.co project
  • Pages are rendered with dynamic data using the Jinja2 templating engine
